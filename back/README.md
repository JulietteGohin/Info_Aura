# Back

Use flask.

## Install

Use python at least 3.8.

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## How-to

```bash
python run.py
# or
flask run
```

Go to [http://127.0.0.1:5000/](http://127.0.0.1:5000/).
