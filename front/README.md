# Front

Use reactjs + nextjs.

## Install

Install Nodejs.

```bash
npm install
# or
yarn install
```

## How-to

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Go to [http://localhost:3000](http://localhost:3000).
